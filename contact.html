Frequently Asked Questions

What are SENTINEL's system requirements?

+

SENTINEL requires a modern web browser with JavaScript enabled, and Python 3.8+ for backend operations. For camera integration, CCTV cameras with RTSP streaming capability are recommended. The system can run on most modern Windows, Linux, or macOS systems with at least 8GB RAM and a dedicated GPU for optimal performance.

How accurate is the facial recognition?

+

Our system achieves 99.7% accuracy under optimal conditions and maintains 95%+ accuracy in challenging conditions (poor lighting, partial face visibility, distance). False positives are minimized through our multi-stage verification process that considers multiple facial features and comparison points.

Is my data secure and private?

+

Yes, SENTINEL employs end-to-end encryption for all data in transit and at rest. All facial data is stored securely in an encrypted database with strict access controls. Our system complies with relevant data protection regulations, and we do not share data with any third parties except as required by law enforcement with proper legal authorization.

How does the alert system work?

+

When a match is detected, SENTINEL automatically sends alerts through your configured channels, which can include SMS messages via Twilio, voice alerts via dwani.ai, and email notifications. These alerts include the match details, confidence score, location, timestamp, and a link to the full detection record in the system.

Can I integrate SENTINEL with existing security systems?

+

Yes, SENTINEL offers integration capabilities with most modern security systems through our API. We can connect to existing CCTV infrastructure, access control systems, and security management platforms. Our team can provide custom integration solutions based on your specific system requirements.

How many cameras can the system monitor simultaneously?

+

The standard SENTINEL configuration can monitor up to 16 camera feeds simultaneously on a recommended hardware setup. For larger installations, our enterprise solution can scale to handle hundreds of cameras across multiple locations using distributed processing architecture.
